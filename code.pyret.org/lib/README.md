This directory has downloaded copies of a few JQuery and Google-related
libraries. They originally had references via CDNs in our source code; those
URLs are included in the relevant subdirectory READMEs. They've been moved to
be in our source tree because:

- Some are getting old and may eventually not be supported as-is
- It's useful to have all of our dependencies on the filesystem for hosting
  without an internet connection and just a local web server
